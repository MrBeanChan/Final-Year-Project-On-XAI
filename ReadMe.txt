				PAITIENT READMISSION USING XAI
---------------------------------------------------------------------------------------------------------
Hardware Requirements:

RAM: 8GB (Recommended)
Storage: 500 MB (SSD if possible)
Clock Speed: 2.0 Ghz or above
CPU : Quad core or more

---------------------------------------------------------------------------------------------------------

Software Requirements:

Python == 3.10 or above
IDE with Jupyter Notebook to run python notebooks
Internet connection to install required packages(if not already pre installed)


The following libraries will be installed if not found but are mentioned to inform on requirement:
streamlit == 0.79.0
pandas == 1.2.3
numpy == 1.18.5
matplotlib == 3.3.2
seaborn == 0.11.0
scikit_learn == 0.24.1


---------------------------------------------------------------------------------------------------------

To just execute project:

Run the "RunMe.py" file using any python IDE.
(It shall install all the required packages and run the GUI)

--------------------------------------------------------------------------------------------------------
To build entire project again:

Run the files in following order
1) 1_Processing.ipynb
2) 2_Binary_Models.ipynb
3) 3_XAI

After which you run the "RunMe.py" file
